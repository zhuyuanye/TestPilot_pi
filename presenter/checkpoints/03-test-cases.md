## 自动化主线

30 分钟演示优先实现：

```text
TC-01 → TC-02 → TC-05 → TC-14 → TC-16
```

差异诊断选择：

```text
TC-04（21 位用户名）或 TC-13（手机号 123）
```

## 数据策略

- 用户名：`tp` + 时间片 + 随机数，总长度不超过 20。
- 手机号：核心流程生成以 1 开头、第二位为 3～9 的 11 位数字，以同时通过当前 Vue 校验；更宽的产品口径单独作为差异测试。
- 每条用例记录自己创建的用户 ID。
- 使用 `finally` 清理数据，不能依赖测试执行顺序。
- 删除不存在用户时先创建对照用户，执行后再次查询对照用户。
