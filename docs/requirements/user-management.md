# 系统用户新增与删除需求

## 1. 业务背景

系统管理员需要通过“系统管理—用户管理”维护系统用户。

## 2. 新增用户

- 用户名必填，长度为 2～20 个字符。
- 用户名在系统中必须唯一。
- 用户昵称必填。
- 密码必填。
- 手机号选填；填写时必须符合合法的手机号码格式。
- 新增成功后，用户应出现在用户列表中。
- 新增用户的默认状态为“正常”。

## 3. 删除用户

- 管理员可以删除普通系统用户。
- 删除成功后，该用户不应继续出现在用户列表中。
- 当前登录用户不能删除自己。
- 删除不存在的用户时，不能影响其他用户。

> 本文档为原始需求，不包含验收标准。需求中未说明的规则不得自行补充。
